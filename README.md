# Book Review Platform
**IT8415 – Database Programming 2 | Group 6**

A data-driven web application built with PHP and MySQL that allows users to browse, create, and manage book reviews.

## GitHub Repository
https://github.com/yusuf-mahdi/IT8415-S1-G6

## Features
- Three user roles: Admin, Content Creator, Viewer
- Book review creation, publishing, and management
- Star rating and comment system
- Advanced search by keyword, date range, creator, and popularity
- Admin dashboard with user management and reports
- AJAX comments and database triggers as advanced features

## Tech Stack
PHP, MySQL, Apache (XAMPP), HTML, CSS, JavaScript
